## 3. 특성공학(Feature Engineering)

1. 날짜
- 평일,주말인지 구별한 열(on_trending_day) 생성
- 인기동영상 유지 기간을 확인할 열(trending_days) 생성
2. 영상 길이 분류 (duration_group)
- 2분이하(쇼츠), 2-10분, 10-20분, 20-40분, 40분-1시간, 1시간이상
3. 태그 개수, 디스크립션 글자수
- 태그 개수(tags_count), 디스크립션 글자수(des_count), 디스크립션 속 해시태그 개수(hash_count), 외국어 글자수(fore_count)
4. 유튜브 채널의 등급 표시 (channel_level)
- orpal(1만미만), bronze(1만~10만), silver(10~100만), gold(100~1000만), diamond(1000만 이상) 구분
5. 인기동영상 선정 후 변화율 확인
- 조회수,구독자,좋아요,싫어요,댓글 백분율로 표기
- view_deriv_per, subscrib_deriv_per, like_deriv_per, dislike_deriv_per, comment_deriv_per
6. 트렌드 확인
- 해당 비디오조회수/전체 조회수 비율(view_ratio)
- 총조회수/ 비디오개수(trend_ratio)
- 좋아요싫어요 비율(like_ratio)
- 업로드날짜/선정날짜 차이(diff_on)
- 업로드 날짜와 인기동영상 선정 일자 비교후 조회수로 나누어 증가 속도 유추 (pace)
- 구독자 대비 조회수 비율 (view_sub)
7.  클러스터링 (cluster)

## 5. 인기동영상 판단 지표(Q2)

### 5-1. 아이디어

- 인기동영상 선정될 때 조회수를 통해 인기동영상 최저 조회수를 확인(최저 : 53297)
- 트렌드 분석
- 영상 외 영향 분석 : 해시태그수, 외국어 글자수 등 영상 외적인 부분을 확인하였을때 크게 관련이 없어보입니다.

### 5-2. 판단 지표

### 1. **트렌드**

트렌드라는 지표는 기존에 많은 구독자를 소유하고 있는 채널이 아닌 신생채널에서 구독자 변화율, 조회수 변화율이 높다면 현재 트렌드임과 동시에 인기동영상의 기준이 조회수,구독자수만이 아니라는 것을 검증하기 위한 지표입니다.

지표의 기준으로는 채널 내 적은 영상 수(10~20개미만), 구독자 변화율, 조회수 변화율이 상위 20%에 속함 입니다.

변화율을 고려하여 유효하다고 생각되는 구독자 수, 조회수 지표를 활용하면서 절대적인 수치보다 유연하게 적용가능합니다.

지표 적용 결과는 먹방(천뚱,햄지), 연예인(임영웅,김종국), 마인크래프트, 아이돌, 감동영상, 예능 등이었습니다. 이는 절대적 수치의 조회수, 구독자 수에 민감하지 않고 작년 트렌드를 잘 찾았습니다.

<br/>

### 2. Pace

 Pace라는 지표는 영상의 업로드 날짜와 인기동영상 선정 일자를 비교하여 해당 값으로 선정일 조회수를 나누어서 현재 데이터에서 확인 불가능한 인기동영상 선정 전의 증가속도를 유추합니다.

![12](https://user-images.githubusercontent.com/87019897/175303713-a82078da-ae29-4709-be19-00f5a0e51cf0.png)


그래프를 보면 하루에 50만뷰를 달성할 정도의 파급력이 있어야 인기동영상에 많이 채택되는 것 같습니다.

평균적으로 1시간에 2만명, 10분에 3500명 가량이 조회해아하는 파급력입니다. 최저점의 경우에는 하루에 18000명 정도 조회하는 속도로도 가능했습니다.

<br/>

### 3. 구독자 대비 조회수

이 지표는 간단하면서도 좋은 지표입니다. 좋아요, 싫어요 지표의 경우에는 조회수의 증가율에 비해 턱없이 증가율이 낮습니다. 그러나 구독자 수 대비 조회수는 해당 영상이 알고리즘의 선택이라고 불리우는 현상을 통해 급격하게 노출되고 있음을 알려줍니다. 

![13](https://user-images.githubusercontent.com/87019897/175303724-46e93c19-3fa3-4973-8433-cf66a50f380b.png)

이 지표는 선정됐을 때 조회수와 구독자수를 비교함에도 불구하고 조회수가 구독자수를 넘는 경우가 많습니다. 100만 유튜버가 영상을 올렸을 때 50만 이하로 조회수가 나오는 것을 보더라도 상당히 높은 수치입니다.
